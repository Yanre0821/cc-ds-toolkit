# 答辩 PPT / Thesis Defense Presentation

> 毕业论文/竞赛答辩演示文稿生成

## 做了什么

从论文/报告文本生成结构化的答辩 PPT，包含：研究背景、方法、结果、讨论、致谢等标准模块。

## 用到的技能

- **PPTX** — 学术答辩 PPT 生成
- **Office Scripts** (辅助) — 批量调整页码、修复格式

## 典型流程

```
论文/报告文本 → 提取关键内容 → 规划幻灯片结构 → 生成 PPT → 格式检查
```

## 关键要求

- 正文字号 ≥ 24pt，标注 ≥ 18pt
- 单页文字不超过 7 行
- 图表占页面面积 ≥ 40%
- 答辩逻辑链清晰：问题→方法→结果→结论
