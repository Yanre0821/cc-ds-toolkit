# CUPT 环形喷泉答辩 / CUPT Ring Fountain Defense

> 中国大学生物理学术竞赛 — 环形喷泉问题答辩 PPT

## 做了什么

针对 CUPT 环形喷泉问题（环形水帘的形成与稳定性），从实验数据、理论分析、仿真结果三方面整合，生成竞赛答辩用演示文稿。

## 用到的技能

- **PPTX** — 答辩 PPT 生成
- **Office Scripts** — DOCX → PPT 内容迁移、格式统一

## 关键数据来源

- 实验测量数据（水帘直径、流速、稳定性判据）
- COMSOL 流体仿真截图
- 理论推导公式（Laplace 压力、Plateau-Rayleigh 不稳定性）
