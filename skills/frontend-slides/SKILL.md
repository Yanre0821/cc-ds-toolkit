# Frontend Slides

> 创建精美的 HTML Web 幻灯片——从零生成或从 PPT 转换。

## 功能 / Capabilities

- **零依赖** — 单 HTML 文件，无框架，打开即用
- **视觉风格发现** — 生成预览让你挑选，不用描述
- **PPT 转换** — .pptx → HTML 网页
- **反 AI 审美** — 精心设计风格，告别紫渐变
- **固定 16:9** — 标准演示比例

## 安装 / Installation

通过 Claude Code 插件市场安装：

```
/plugin marketplace add https://github.com/zarazhangrui/frontend-slides
```

## 使用示例 / Usage

```
用 Frontend Slides 做一个项目介绍网页
把这个答辩 PPT 转成网页版
换一个更简约的风格
```
