# Humanizer-zh — 中文去 AI 味

> 检测并修复中文文本中的 AI 生成痕迹，让文字读起来像人写的。

## 功能 / Capabilities

自动检测并修复以下 AI 写作特征：

- 夸大的象征意义和宣传性语言
- 以 "-ing" 结尾的肤浅分析
- 模糊归因（"相关研究表明""许多人认为"）
- 破折号过度使用
- 三段式法则（"第一...第二...第三..."）
- AI 高频词汇（"深刻""彰显""赋能""重塑"）
- 否定式排比
- 过多的连接性短语

## 使用场景 / When to Use

- 编辑 AI 辅助写的论文/报告
- 润色答辩讲稿
- 让竞赛申报书写得更自然

## 使用示例 / Usage

```
帮我把这段文字去一下 AI 味
这篇报告帮我过一遍，去掉 AI 痕迹
```
