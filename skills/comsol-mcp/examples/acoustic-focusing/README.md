# CUPT 同心圆环声聚焦 / Acoustic Focusing with Concentric Rings

> 中国大学生物理学术竞赛 (CUPT) — 声场聚焦仿真

## 做了什么

用 COMSOL 对同心圆环结构进行声场仿真，研究几何参数（环间距、环宽度、环数）对聚焦效果的影响。

## 用到的技能

- **COMSOL MCP** — 模型参数化、求解控制、结果读取
- **Vibe Simulation** — 批量参数扫描

## 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 圆环材料 | 铝 / 钢 | 对比声阻抗影响 |
| 频率范围 | 2-20 kHz | 可听声频段 |
| 几何变量 | 间距 / 宽度 / 环数 | 参数化扫描 |

## 输出

- 声压场分布图
- 聚焦增益 vs 频率曲线
- CUPT 竞赛汇报 PPT
